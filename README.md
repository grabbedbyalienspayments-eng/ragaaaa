# Black Studio

Website one-page pentru Black Studio - Training și Management Modele Videochat OnlyFans.

## Deploy pe Vercel

### Pasul 1: Pregătire
```bash
npm install
npm run build
```

### Pasul 2: Deploy
1. Conectează repository-ul la Vercel
2. Setează Build Command: `npm run build`
3. Setează Output Directory: `dist`
4. Deploy automat

### Pasul 3: Configurare Domeniu
- Adaugă domeniul `black-studio.com` în setările Vercel
- Actualizează DNS-ul domeniului cu recordurile Vercel

## Structură Proiect

```
├── public/           # Assets statice (favicon, robots.txt, sitemap.xml)
├── src/
│   ├── pages/       # Componente pagină
│   ├── i18n/        # Traduceri RO/EN
│   └── router/      # Configurare routing
├── vercel.json      # Configurare Vercel
└── package.json
```
